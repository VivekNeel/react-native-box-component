# react-native-box-component

A box component for react-native.

# Installation

### Using npm :

`npm install react-native-box-component`

### Using yarn :

`yarn add react-native-box-component`

# Usage :

```import Box from "react-native-box-component";

const App = () => {
  return <Box
  flexDirection="row"
  justifyContent="center"
  alignItems="center"
  marginTop={16}
  >{children}</Box>;
};
```
